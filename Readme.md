<h2>Usage</h2>
First clonning repository using

```bash
git clone https://github.com/Brspontes/todolist-react.git

cd todolist-react
```
<br />
In current folder, before compilation, restore packages with

```bash
npm install
or
yarn install
```
Run App
```bash
npm run dev
or
yarn dev
```

Go to:  https://localhost:8080/

app opnend
